 📚 Linguagem de Programação
--
Repositório da disciplina de Linguagem de Programação, contendo exercícios, projetos, anotações e atividades desenvolvidas ao longo do curso.

🎯 Objetivo da Disciplina

A disciplina tem como foco o desenvolvimento da lógica de programação e a construção de algoritmos utilizando as linguagens C# e C++. Durante as aulas, são trabalhados conceitos fundamentais da programação estruturada e boas práticas de desenvolvimento.

🛠 Tecnologias Utilizadas

 C#
- C++
- Git
- GitHub
- Visual Studio Code


📖 Conteúdos Estudados

Até o momento, os conteúdos abordados foram:

- Estruturas de seleção
 - if
 - else
 - switch

- Estruturas de repetição
 - for
 - while
 - do while

- Funções
 - Criação de funções
 - Parâmetros
 - Retorno de valores

- Escopo de variáveis
 - Variáveis locais
 - Variáveis globais
 - Tempo de vida das variáveis
